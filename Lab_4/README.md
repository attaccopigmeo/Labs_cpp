# Лабораторная работа № 4 (Шмидт Камилла, ИТ-7, Вариант - 10)

## Задание 1 - Triangle

### Условие - 1

Реализовать определение класса (поля, свойства, конструкторы (не менее трех),
перегрузку оператора вывода для вывода полей, заданный метод согласно варианту).
Протестировать все методы, включая конструкторы, все исходные данные вводятся с клавиатуры, не
забывайте проверять данные на корректность. В главной функции main показать работу всех функций с дружественным интерфейсом.
Поля:
`double a`
`double b`
`double c`

Определить, существует ли треугольник с
такими длинами сторон. Результат
должен быть типа `bool`.

### Решение - 1

Класс включает три конструктора: по умолчанию (создает треугольник с нулевыми сторонами), с параметрами (инициализирует стороны заданными значениями), копирования (новый объект создается на основе уже существующего). Для работы с полями реализованы сеттеры и геттеры, обеспечивающие безопасное чтение и изменение значений сторон. Метод `exists()` проверяет возможность существования треугольника с заданными сторонами по неравенству. Перегрузка оператора `<<` позволяет удобно выводить стороны треугольника на экран.

### Тесты - 1

```Введите сторону a: 2
Введите сторону b: 6
Введите сторону c: 8
Треугольник не существует.
t2 (по умолчанию): Стороны треугольника: a = 0, b = 0, c = 0
t2 после установки значений: Стороны треугольника: a = 3, b = 4, c = 5
Треугольник существует.
t3 (копия t1): Стороны треугольника: a = 2, b = 6, c = 8
```

## Задание 2 - Triangle

### Условие - 2

Добавить к реализованному в первом задании классу указанные в варианте
перегруженные операции. В главной функции main показать работу всех функций с дружественным интерфейсом.
Унарные операции:

- вычисляется площадь треугольника, результатом должно быть вещественное число.

Операции приведения типа:

- `double` (неявная) - результатом является периметр треугольника;
- `bool` (явная) - результатом является `true`, если треугольник с такими длинами сторон существует, и `false` в противном случае.

Бинарные операции:

- `<` Triangle t - сравниваются площади треугольников;
- `>` Triangle t - сравниваются площади треугольников.

### Решение - 2

К классу из 1-й задачи добавлено:

- приватный метод `double area() const`, возвращающий площадь по формуле Герона. Если треугольник не существует, то выводим 0.0;
- унарный оператор `double operator~() const`, используется для удобного получения площади через `~t1`, применение - пример перегрузки операторов;
- приведение типов, неявное приведение к double возвращает периметр треугольника, а явное приведение к `bool` возвращает, существует ли треугольник;
- операторы сравнения (сравнивает треугольники между собой по площади);

### Тесты - 2

```Введите сторону a: 6
Введите сторону b: 7
Введите сторону c: 8

Треугольник t1: a = 6, b = 7, c = 8
Существует: Да
Периметр: 21
Площадь: 20.3332

Введите сторону a: 8
Введите сторону b: 9
Введите сторону c: 2

Треугольник t2: a = 8, b = 9, c = 2
Существует: Да
Периметр: 19
Площадь: 7.3101

t1 имеет большую площадь, чем t2.

t3: a = 3, b = 4, c = 5 Площадь: 6 Периметр: 12
t3 существует? Да
```
